üí° The Core Insight
HFT competes on latency, not uptime.

In HFT, the game is: "Can I see the price and react before anyone else?"

If you add 1ms to every order, you're 1ms behind on every single race. In arbitrage, the first one to react wins. Being 1ms slower means you lose every time.

But if you're down for 5 seconds once a month? You miss a few races entirely ‚Äî but you win all the other races because you're the fastest.

Think of it like a sprinter:
Option A: Run every race with a 10kg weight (always slower)
Option B: Skip 1 race per month, run all others at full speed

Option B wins more races overall, even though it misses a few.
